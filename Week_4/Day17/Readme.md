# Day 17 - Internship Journey

## Overview
Day 17 of my internship, focused on basics of fast api
## Topics Covered
- GET, PUT, POST requests
- Query Parameters
- Pydantic Models

## Key Learnings
- GET request is used to fetch information from server/API
- PUT request is used to give information to server/API
- POST request is also used to give information to server/API the only difference being PUT is used to update or replace an existing resource at a known location.

## Notes
- API routes are not functions they are HTTP entry points.
