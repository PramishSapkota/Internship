## Key Learnings
- **iterator** is an object that contains a countable number of values. Two main functions are iter() and next().

- **generator** is similar to a function but with 'yield' instead of 'return'. it also has iter() and next(). 

- **zip** is used to combine two or more iterables

- **map** is used to executes a specified function for each item in an iterable.

- **filter**is similar to map but the function in filter should return True or False.

- **enumerate**
without it we would have to manually use and increment a variable for indexing. But enumerate gives us a index on the loop  

- **tuple unpacking**
Its a concept to pack and unpack the elements of tuple.

- **asynchronous functions**: It is a concept to run the functions asynchronously
