# Day 2 - Internship Journey

## Overview
Day 2 of my internship, focusing on loops and data types of python.

## Topics Covered
- Loops (for, while statements)
- Data types (list, tuple, set, dictionary)
- Reversing of a string using loops and checking for palindrome numbers

## Key Learnings
- Lists are ordered, mutable and allows duplicate entry.
- Tuples are ordered, non-mutable and allows duplicate entry.
- Sets are un-ordered, unchangeable and doesn't allow duplicate entry. 
- Dictionary are ordered, mutable and doesn't allow duplicate entry.
